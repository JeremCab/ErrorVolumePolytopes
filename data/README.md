# Data Directory

## Original sample pools

Subsets of FashionMNIST correctly classified by the full-precision model.

| File | Samples | Notes |
|------|---------|-------|
| `fashionMNIST_correct_mlp.pt` | 56,680 | Correctly classified by fp MLP |
| `fashionMNIST_correct_cnn.pt` | 59,297 | Correctly classified by fp CNN |

## First-generation augmented datasets

Early augmentation experiments (200 walk steps per seed, no q-correct/q-incorrect split).
Kept for reference.

| File | Samples | Notes |
|------|---------|-------|
| `fashionMNIST_augmented_mlp_seed42.pt` | 2,000 | First-gen augmentation, MLP |
| `fashionMNIST_augmented_cnn_seed42.pt` | 2,000 | First-gen augmentation, CNN |

## Selected seed points (x₀)

10 original samples per group, used as starting points for data augmentation.
All are correctly classified by the fp model; split by 4-bit q-model classification.

| File | Samples | Notes |
|------|---------|-------|
| `qmodel_qcorrect_mlp_b4.pt` | 10 | MLP seeds, correctly classified by 4-bit q-model |
| `qmodel_qincorrect_mlp_b4.pt` | 10 | MLP seeds, incorrectly classified by 4-bit q-model |
| `qmodel_qcorrect_cnn_b4.pt` | 10 | CNN seeds, correctly classified by 4-bit q-model |
| `qmodel_qincorrect_cnn_b4.pt` | 10 | CNN seeds, incorrectly classified by 4-bit q-model |

## Current augmented datasets

Canonical datasets for the experiments: 10 seed points × 10 augmented points each.
Augmented points are generated by random walk within P1; they inherit label c from x₀.

| File | Samples | Notes |
|------|---------|-------|
| `fashionMNIST_augmented_mlp_seed42_walk_qcorrect.pt` | 100 | MLP, q-correct group |
| `fashionMNIST_augmented_mlp_seed42_walk_qincorrect.pt` | 100 | MLP, q-incorrect group |
| `fashionMNIST_augmented_cnn_seed42_walk_qcorrect.pt` | 100 | CNN, q-correct group |
| `fashionMNIST_augmented_cnn_seed42_walk_qincorrect.pt` | 100 | CNN, q-incorrect group |
